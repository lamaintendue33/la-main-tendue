Configurer l'envoi d'emails

Pour que le formulaire de contact du site fonctionne enfin, il te faut créer un compte gratuit sur un service qui se charge d'envoyer les emails. Ça prend environ 10 minutes, suis simplement les étapes ci-dessous.

1
Créer ton compte
Sur le site resend.com — gratuit, aucune carte bancaire demandée
  • Va sur resend.com
  • Clique sur le bouton « Sign Up » en haut à droite
  • Inscris-toi avec l'adresse lamaintendue33@gmail.com
  • Si un e-mail de confirmation arrive dans cette boîte, clique sur le lien à l'intérieur
Pourquoi cette adresse et pas une autre ? Comme ça le compte appartient à l'association, pas à une personne en particulier — plus simple à gérer sur le long terme.
2
Ajouter le nom de domaine du site
Pour que les emails partent bien depuis « lamaintendue33.fr »
  • Une fois connecté(e), clique sur « Domains » dans le menu à gauche
  • Clique sur le bouton « Add Domain »
  • Dans le champ, tape exactement : lamaintendue33.fr
  • Clique sur « Add »
Un tableau va apparaître avec des lignes techniques (DNS) Ne touche à rien toi-même ici — fais une capture d'écran de ce tableau et envoie-la moi. Je m'occupe ensuite d'ajouter ces lignes sur OVH, exactement comme on l'a fait ensemble pour connecter le nom de domaine au site.
📸 Capture d'écran à envoyer : le tableau complet avec les colonnes Type / Nom / Valeur.
3
Attendre la vérification
Généralement automatique en quelques minutes, parfois jusqu'à 24h
  • Une fois que j'ai ajouté les lignes DNS sur OVH, retourne sur la page « Domains » de Resend
  • Le statut à côté de lamaintendue33.fr doit passer de « Not Verified » à « Verified » (généralement avec une coche verte)
  • Si ça reste bloqué après plusieurs heures, préviens-moi
4
Créer la clé secrète
C'est la dernière étape — cette clé permet au site d'envoyer des emails
  • Clique sur « API Keys » dans le menu à gauche
  • Clique sur « Create API Key »
  • Donne-lui un nom, par exemple : Site La Main Tendue
  • Clique sur « Add »
  • Une clé va s'afficher, elle commence par re_... — copie-la tout de suite, elle ne sera plus jamais visible ensuite
⚠️ Cette clé est comme un mot de passe Ne l'envoie jamais par SMS ou message classique en clair. Le plus simple : donne-la-moi de vive voix ce week-end, ou envoie-la par un message qui s'autodétruit.

Une fois que j'ai la clé

Je l'ajoute dans les réglages du site (Vercel) en 2 minutes, et le formulaire de contact fonctionnera enfin : chaque message envoyé depuis le site arrivera directement dans lamaintendue33@gmail.com.

Une question, un blocage sur une étape ? Écris-moi, je réponds vite.
